# Validator Configuration for ZUGFeRD

This repo contains a `scenarios.xml` file for use with the [KoSIT Validator](https://github.com/itplr-kosit/validator) to validate ZUGFeRD 2.5 / Factur-X 1.09 XML files.

Also included is a `scenarios_zugferd1.xml` to validate legacy ZUGFeRD 1.0 XML files.

It requires the [KoSIT Validator](https://github.com/itplr-kosit/validator) to execute the rules and uses the official [ZUGFeRD releases](https://www.ferd-net.de/standards/zugferd). 
Visit the [Validator documentation](https://github.com/itplr-kosit/validator/blob/main/docs/cli.md) to learn how to use this.
